# sb

A simple personal website template with blog functionality.

## Usage

```console
$ git clone https://github.com/vilhelmbergsoe/sb
$ cd ./sb/tools/
$ ./createuser admin pass && cd ../
$ go get
$ go run .
server running on :8080
```

## Endpoints

`/` home page

`/admin` admin panel
